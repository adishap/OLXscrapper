#OLXscrapper
This is small script of python which uses "Beautiful Soup" library and Python3 for web scrapping from website olx.in.
We can search products from multiple categories and from diffrent states whose price is from low to high.

The result is obtained in form of a csv file with given product specification:
1: Title
2: Product Category and Company
3: City
4: Price
5:Link to the product
